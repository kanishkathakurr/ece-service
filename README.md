# ECE Pipeline Microservice

Reusable Node.js + TypeScript microservice for document ingestion, parsing, chunking, embedding, vector storage, retrieval, source metadata, and generic extraction.

## Features

- Document Upload
- Document Status Tracking
- Parsing
- Chunking
- Embedding Generation
- Qdrant Vector Storage
- Search / Retrieval
- Source Metadata
- Generic Extraction
- Document Deletion
- Health Check

## Tech Stack

- Node.js
- TypeScript
- Express
- Zod
- Qdrant
- Vercel AI SDK
- Docker Compose

## Setup

```bash
npm install
docker compose up -d
npm run dev
